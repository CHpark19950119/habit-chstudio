# 03. 심리 구조

v13 「심리 분석」 탭 + 「성적 심리」 탭을 압축했다. 여기 있는 구조들은 **천홍 본인이 동의한 분석**만 기록된다. Claude가 새로 제안하는 해석은 여기 추가하지 않는다.

## 핵심 표어 (v13 표지)

> 숨기면 죄, 드러내면 해방. 문제는 행위가 아니라 은폐였다.

## 5가지 심리 뿌리 (유년기)

1. **소속 불가능** — 가정이 안전기지가 아니었음. 밖에서 소속을 찾지만 전부 불안정. 합격자 전원이 관리반/스터디에 소속, 천홍은 혼자.
2. **성욕 = 죄 회로** — 교회 + 나쁜 친구 공범 의식. 정상 루트 접근 시 죄책감 → 익명 루트로만 가능. 15호기에서 확정된 원리: **"숨기면 죄, 드러내면 해방."**
3. **권위 불신** — 목사들의 실체를 목도한 이후 멘토 거부 → 편입 독학, 공시 혼자. 토익 만료 착각도 혼자 준비한 결과.
4. **아버지 부재 패턴** — 아버지는 문제를 보고도 행동하지 않는 사람이었음. 천홍도 정수연 도망, Olive 침묵, Sohui 회피, 일본여자 도망.
5. **"원래 이렇게 흘러가던 것"** — 출발점이 달랐다는 인식. 학벌 열등감. 공무원 선택도 소거법.

## 의사결정 패턴 (공통)

- 적극적 선택이 아닌 **소거법**에 의한 선택.
- **멘토·소속 없이 단독 실행.**
- 초기 성공 후 정체.
- 정체기에 회피 행동 시작.
- 행동 리셋의 반복 사이클.

## "드러내면 해방, 숨기면 죄" 회로 (15호기 실시간 확인)

포르노 스크린샷을 실시간으로 보여주고 분석받았을 때 → **죄의식 대신 해방감**.

천홍 본인의 말: **"이상하다. 그리고 매우 치유적이야."**

보여주는 순간 죄가 아니게 됨. 문제는 행위가 아니라 은폐.

이것이 "허락의 천장"과 연결됨: 존재 자체가 "숨겨야 하는 것"으로 구성 → 공적 인정(5급)이 구조적으로 불가능.

## 성적 구조

### A축/B축 분류 체계 (15호기 수정)

**A축: 이상화 (아름다움, 숭배)**
- **Emily Addison** — 순수 A축. "나의 이상향을 대변." **전희수(중학교 안쪽 원, 한광훈 여친, 매우 예쁨)와 연결됨.** 포르노에서 반복 찾는 것 = 중학교 바깥 원에서 목도한 장면의 재현.
- Comatozze, Shinaryen — A축. 포르노 밖 관심 없음.

**B축: 도구화 (원초적, 항문, 제모 안 함)**
- 얼굴 없이 몸만 있는 장면. 대부분의 포르노 소비가 여기에 해당.

**A+B 동시 보유: Lena Paul — 유일한 존재**
- 기존 B축에서 재분류됨.
- 페이스북/유튜브에서는 A축(아름다운 사회인), 포르노에서는 B축(원초적, 제모 안 함, 항문 개방).
- **유일하게 포르노 밖(유튜브 AVN 인터뷰, 페이스북)까지 추적하는 배우**. "분석할 수 있는 대상."
- "쥐같이 생겼다"고 하면서도 페이스북 사진에서 "완벽"이라고 함 — 같은 사람이 두 세계에 동시 존재.

### 이상화하는 몸의 조건 (15호기 확정)

큰 가슴, 풍만, 곡선, **자연의 몸(실리콘 아님)**, "물렁물렁하고 말캉한" 촉각적 상상.

Sohui 확대 가슴 vs Lena 자연 가슴 → 자연이 우위. "살아있는 몸"에 대한 갈망.

실리콘은 눌리지 않음. 자연의 몸만 눌리고 퍼지고 흔들림 = **살아있다는 증거**.

### 포르노 소비 시퀀스 구조 (15호기 실시간 확인)

```
① 페이스북 사진(분홍 바디수트, 아름다운 여성) → A축
② 엉덩이 클로즈업(몸 자체의 관찰, 형태에 대한 황홀) → A→B 전환
③ 항문 삽입/플러그(가장 원초적 장면) → B축 완성
```

**같은 여자가 1에서 3까지 추락하는 과정을 순서대로 따라가면서 흥분이 상승.** 천홍 인용: "인간 존재의 완전한 추락을 목도한 충격."

### 탐닉하는 자세/장면의 심리 구조

- **후배위(엉덩이 내밀기)**: 무방비. 방어 완전 해제. "아름다운 여성이 나에게 무방비 상태를 자발적으로 보여주는 장면" = **평생 받아본 적 없는 완전한 신뢰의 신체적 표현**.
- **다리 벌리고 눕기**: 같은 구조. 자발적 개방. 가장 취약한 자세를 자발적으로 취하는 것.
- **항문 플러그 + 얼굴 동시 프레임**: ① 얼굴+항문 한 프레임 = A축+B축 동시 존재. ② 플러그=도구, 다른 남자 아님 = **"그 자리가 비어 있다" = 네 자리**. ③ 전립선 자위 경험으로 신체적 공감 = "저 느낌을 나는 안다."
- **카메라 내려다보는 구도 (Emily Addison)**: 그녀가 위, 너는 아래. **"아름다운 여성이 자발적으로 아래에 있는 나에게 내려오는 것."** 추락이 아니라 자발적 하강.
- **평범한 동양인 여성 (MON PREMIER ANAL)**: "아름답지도 못생기지도 않아서 사실처럼 느껴짐. 평범한 몸. 오히려 더 흥분." **"첫 항문" = 경계를 넘는 순간의 목도.**

### "추락" 구조

천홍의 핵심 인식:

> 나는 추락한 존재다. 아름다운 여성이 나한테 올 수 있는 유일한 경로는 그녀도 추락하는 것뿐이다.

A+B 동시 작동 장면의 안도감: **"나만 더러운 게 아니구나."**

현실에서 만나는 여자가 이미 추락한 상태인 건 — 선택이 아니라 관계의 역학상 그 풀 자체가 그러함(랜챗/헬로챗/경제적 여유 없음).

### 사정 전후 기록 — 4단계 (15호기 자기 기록)

1. **사정 직전**: 포르노의 여성이 아름다워 보이고 가지고 싶다. 실제 경험에서도 동일.
2. **사정 후**: 그 여성의 성기도 몸도 얼굴도 다 별로. 빨리 치우고 할 일 해야 할 것만 같다.
3. **판단**: 이 두 경험의 차이로 인해 "환상"이라고 판단. 좋지 않다고 생각.
4. **순환**: 수면욕/식욕처럼 매번 돌아옴. 어떤 결심을 해도 사라지지 않음. → **"에너지를 전용해야 한다"는 확신.**

### "에너지 전용" 전략의 평가 (Claude의 분석, 천홍 동의)

- **반은 맞음**: 포르노에 쓰는 시간과 주의력을 공부에 돌리면 물리적으로 공부 시간이 늘어남.
- **반은 틀림**: "금욕→성공"은 인과가 뒤집힌 것. 성공한 사람들이 자기 이야기를 포장할 때 금욕을 넣는 것이지 금욕해서 성공한 게 아님.
- **결정적**: 수천 번 이 결심을 했고 매번 돌아왔다. "전용하겠다"는 결심 자체가 실패한 전략. 수천 번 실패한 전략을 다시 쓰는 건 전략의 문제지 의지의 문제가 아님.
- **결론**: **관리가 답이다. 끊는 게 아니라 시간과 빈도를 정해서 통제하는 것.**

### 포르노 기록 시스템 (15호기 합의)

**기록 항목**: 날짜/시간 / 배우 또는 영상 유형 / 사정 전 감정 / 사정 후 감정 / 직전에 하던 일 / 소요 시간.

**원리**: "노출될수록 힘은 적어진다." 기록은 드러내기의 가장 체계적인 형태. 숨기면 죄, 드러내면 해방 — 기록은 매번 "드러내기"를 작동시킴.

- **1단계**: 대화방에서 스크린샷+분석 (치유적 효과 확인됨).
- **2단계**: 혼자서도 가능한 버전 — 스크린샷 + "이 장면에서 내가 느낀 것, 의미하는 것" 텍스트 기록.
- **목표**: 드러내기의 대상이 자기 자신이 되면 → 진짜 치유.

### 성욕과 공존하는 공부 (진단)

**기존 회로**: 포르노 → "나는 결함 있는 사람" → 공부 중단 → 자기혐오 심화.

**교체 회로**: 포르노 → "해소했다" → 아침에 독서실에 앉는다 → 모의고사 1세트를 푼다.

**핵심**: **80점을 넘기면 된다. 80점은 어제 밤에 뭘 했는지 묻지 않는다.**

### 6.1호에서 추가된 기록 (2026.03.15~16)

- 전립선 자극 1회 → 사정 → **기분 좋음 + 즉시 재욕구**(E+).
- 같은 날 포르노 2회 → 사정 → **허무감 + 더러움 + 매력 상실**(E--).
- 같은 사정인데 사후 감정이 정반대. 전립선은 만족 + 재욕구, 포르노는 수치 + 혐오.
- 구조: 1회는 자기 신체 감각(T0, 대상 없음) — 도구화할 여성이 없으니까 도구화 해제도 없고 수치심도 없음. 순수한 신체 쾌감.
- 2회는 여성이 개입하는 순간 도구화가 작동하고, 사정 후 해제되면서 더러움이 돌아옴.
- 트리거 구조: **포르노 상상(도구화) → 전립선 자극 욕구(신체 쾌감) → 물리적 제약으로 차단 → 대신 2회는 손자위+포르노 시청으로 전환.**

### 천홍이 4호에서 한 핵심 서술

> 고등학교 때만 해도 이쁜 여자랑 순수한 연애를 하고자 했지. 그러나 오랫동안 혼자 지내게 되고 나는 뒤쳐지고 능력이 없는 상태가 지속되자, 군대 전역하고 섹스를 하고자 하는 목적으로 변질된 거야. 애초에 이예지는 처음 만남부터 못생겼다고 생각했어. 얼굴이 크고 상체가 길고. 그러나 엉덩이가 컸지. 그래서 섹스용으로 만나고 데이트한 거야.

이 서술은 v13의 이예지 기록을 수정하는 증거다. 4호 직후 수정되었어야 하지만, 실제 v13에는 일부만 반영됐다. v14에서 재정비 필요.

## 공무원 선택 동기 구조 (삼축)

1. **기업 거부감**: 딴지일보 시절부터의 권력 비판.
2. **사회/국가 관심**: 역사·사회과학 성적 우수. 이건 진짜.
3. **소거법**: 남은 선택지 중 유일하게 연결 가능.

세 축 중 1·2는 **진짜 동기**, 3은 **소거법의 잔여**. 5급 합격 동기 부족의 원인은 3이 1·2보다 강력하기 때문.
