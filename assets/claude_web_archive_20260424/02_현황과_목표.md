### 5급 정체의 이중 진단

**표면 진단: 방법론 문제**
- 인강만 들음. 관리반/스터디 없음. 완전히 혼자.
- 합격자 전원이 관리반/스터디에 소속됨 — 천홍은 혼자.
- PSAT은 대학 학점과 다른 종류의 시험. 구체적 전략 부재.

**심층 진단: "허락의 천장"**
- 5급을 준비할 자격이 없다는 감각.
- "숨기는 존재가 가장 공적인 자리에 앉는 것의 불가능성."
- 능력의 문제가 아님: 7급은 1년 만에 70→90(+20). 5급은 2년 연속 70 정체.
- **5급이라는 상징에 대한 내면의 거부.**

**구조적 진단**
- 31년간 "구조 안에서의 자유"만 행사함. **구조 자체를 깬 적이 한 번도 없음.**
- 5급 합격 = 구조를 깨는 것. 편입도 "학교 구조" 안에서 이동한 것일 뿐.
- 구조를 깨본 경험이 없는 사람이 구조를 깨는 시험을 치고 있는 것.

### 6.1호에서 천홍이 직접 한 진단 (2026.03.16)

> "나는 아직 5급을 공부하는 사람이 아니다."

Claude가 물었다: "뭐가 부족해서 아직 그 모드에 못 들어갔다고 느껴? 동기? 방법? 자격 감각?"

천홍: **"동기, 그리고 방법, 감각까지 전부 다."**

→ 이후 세션에서 이 세 축을 분해하는 작업이 미완료 상태.
→ 환경 문제(정다영이 집중 환경을 구조적으로 파괴함)와는 별도의 축이다. 환경만 바꿔서 해결될 문제가 아님.

### 정다영 환경 변수

- PSAT은 고정 장소 + 일정한 루틴 + 방해 차단이 기본 조건.
- 정다영은 다른 공간에 있어도 영역을 넘나들고자 함. 관계 중독 구조 추정(천홍 본인의 분석).
- 정다영 분리는 경제적으로도 정서적으로도 불가능.
- "떨어져 있어도 관계는 안전하다"는 감각을 정다영에게 제공할 수 있는가 — 미해결.

## 내부 목표: 소설 「허락」 완결

### 목적 (천홍 본인 인용, 6.1호)

> 궁극 목표는 「허락」이라는 소설을 완결하는 것이고, 소설의 목적은 나의 인생 경로 확인과 나를 객관적으로 보는 것이다. 그리고 나는 궁극적으로 이 작업을 통해 내가 해야 할 것이 무엇인지 정확히 볼 수 있길 원한다.

### 구조 (22챕터 3부)

**제1부: 흙 (1995~2018) — 11챕터 HTML 완성**
- 01. 754 / 02. 이사 / 03. 사마귀 / 04. 찬양 / 05. 바닥 — `허락_전반_01-05.html`
- 06. 바깥 원 / 07. 목격 / 08. MB / 09. 딴지 / 10. 잉꼬 / 11. 수능 — `허락_후반_06-11.html`

**제2부: 유리 (2019~2023) — 콘티 확정, 집필 대기**
- 12. 소속 — 외대 첫날. "소속되면 잘될 줄 알았다."
- 13. Olive — 영국 3주. 먼저 제안 못함.
- 14. Sohui — 쉐라톤 통유리 후배위. 이별 → PC방 월 55회.
- 15. 이문동 — 낮=커플, 밤=마취.
- 16. 앵무새 — 럭키, 애쉬, 벨라. "아팠던 놈들이어서 싼 가격에 고쳐주려고."
- 17. 야놀자 18,725원 — [시간역순] 정다영 첫 성관계. "섹스를 공유하는."

**제3부: 허락 (2024~) — 콘티 확정**
- 18. 70점 — 첫 시험. 또 70점. 7급은 올랐다. 5급은 움직이지 않는다.
- 19. 그들의 수기 — 합격수기를 읽는다. 데이터를 겹친다.
- 20. 근본적 물음 — "나의 성문제를 보았을 때…" 포르노 시퀀스(페이스북→엉덩이→항문)와 공시 준비의 병행.
- 21. 구조 — "구조 안에서의 자유"를 인식하는 챕터. 31년간의 패턴을 본다.
- 22. 허락 — "나 같은 사람이 5급을 감히"가 아니라 "나의 경로에서 80점을 넘기는가." **마지막 문장은 아직 쓰지 않는다.**

### 문체 규약

- 1인칭 "나".
- 미화 없음, 변호 없음.
- 과거시제 기본.
- 짧은 단문 + 리듬감.
- 가족은 "아버지", "어머니"로 호명. 이름은 문맥상 필요할 때만.

### 소설 집필의 부작용으로 발견되는 것들

소설을 쓰면서 기억이 추가로 복원된다. 4호 세션에서 이예지의 실제 이별 원인(다른 여자 연락 발각)이 처음으로 나왔다. 정수연이 정상 루트 거부의 기원이라는 구조도 소설 작업 중 확정됐다. **소설 집필 = 기억 복원 인터뷰**이기도 하다.

- **다중 페이지로 분리** — 단일 HTML 677줄은 한계. 페이지별 분리 + 링크 구조.
- **데이터 시각화** — 거래내역·성적 기록 등 숫자 데이터를 그래프로.

- **통합 타임라인 시각자료 필수** — 시기별 공적 위치 / 연애 / 거주 / 경제를 한 화면에서 동시 추적 가능해야 함. v14 위젯 초안 존재(1995–2026, 슬라이더+다축), stat 값 직접 조정 필요.
